# チューサポ - 開発状況報告書

## 📋 プロジェクト概要

**プロジェクト名**: チューサポ  
**技術スタック**: React + TypeScript + Tailwind CSS + Vite  
**開発期間**: 2025年6月〜現在  
**プロジェクト種別**: 医療従事者向け穿刺技術相談プラットフォーム  

## 🎯 アプリケーションの目的

「注射や採血、ルート確保などの穿刺で悩んだ時に、このサイトに来れば悩みが解決できる」という本質的価値に特化した医療従事者向けプラットフォーム。

### 対象ユーザー
- 看護師
- 医師
- 臨床検査技師
- その他医療従事者

### 解決する課題
- 穿刺技術の習得困難
- 経験豊富な先輩への相談機会不足
- 実践的なアドバイスの入手困難
- 失敗例から学ぶ機会の不足

## 🧾 現在のソフトウェアの説明（2025年12月時点）

「チューサポ」は、**穿刺（注射・採血・ルート確保・動脈穿刺）に関する「相談」と「学習」**を1つに統合した、医療従事者向けのWebアプリです。

- **相談（Q&A）**: 現場で困った穿刺のケースを投稿し、経験者から実践的な回答を得られます（カテゴリー/検索/ベストアンサー/感謝など）。
- **学習（学習クイズ）**: 書籍『ハヤピンpresents 採血・静脈ルート確保』の内容をもとに、**Lesson別の4択クイズ**で学習できます。回答後は**全問解説（参照ページ付き）**が表示されます。
- **成績の見える化**: 正答率チャートに加え、**週間ヒートマップ**・**XP/レベル**・**連続学習**・**苦手Lesson TOP3** などで、継続しやすい設計です。
- **保存方式（現状）**: 学習クイズの成績は端末内（ブラウザのlocalStorage）に保存され、Lesson完走時に自動更新されます（端末間同期は将来拡張で対応可能）。

## 📋 まとめ

「チューサポ」は、医療従事者の実際のニーズに基づいて設計された、実用性の高いプラットフォームです。技術的には最新のフロントエンド技術を活用し、ユーザビリティと保守性を両立させています。

現在はフロントエンド機能が完成しており、バックエンド開発とより高度な機能の実装が次のステップとなります。医療従事者からの継続的なフィードバックを受けながら、実際の臨床現場で役立つプラットフォームとして成長させていく予定です。
